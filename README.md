# seiformularios
Integração de formulários para o sistema SEI! do Tribunal Regional Federal da quarta região, sem prejuízos para o usuário que não utilizar a extensão

A referida extensão tem como objeto alterar o editor de modelos, de textos padrão e de documentos do Sistema Eletrônico de Informações (SEI!) (https://www.trf4.jus.br/trf4/controlador.php?acao=pagina_visualizar&id_pagina=740)  para permitir a inclusão de formulários no sistema, de forma não invasiva. Para que o uso da extensão seja possível pelos usuários de uma instância do sistema, é necessário que os responsáveis pelos modelos os adequem para tal, através da própria extensão.

Descrição geral:
----------------

A extensão faz pré-processamento de informações, criando espaços reservados com informações adicionais nos modelos de documento, que possibilitarão que o usuário tenha a experiência de preencher um formulário, de forma a permitir a edição apenas desses campos demarcados pelo responsável pelo modelo. O conceito foi desenvolvido com a intenção de criar infra-estrutura não invasiva, invisivel e acessória para criar formulários no sistema "SEI!" e, secundariamente, demonstrar viabilidade de, em futuro próximo, patrocinar o desenvolvimento dos devidos módulos para o sistema já referido com funcionalidade compatível e similar.
